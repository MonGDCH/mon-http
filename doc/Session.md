# Session支持


### 设置session, 支持.二级设置

```php
set(string $key, $value = null): Session
```

#### 参数说明

| 参数名 | 类型 | 是否必须 | 描述 | 默认值 |
| ------------ | ------------ | ------------ | ------------ | ------------ |
| key | string | 是  | 键名 |  |
| value | mixed | 否 | 键值 |  |


### 获取session值，支持.无限级获取值

```php
get(string $name = '', $default = null)
```

#### 参数说明

| 参数名 | 类型 | 是否必须 | 描述 | 默认值 |
| ------------ | ------------ | ------------ | ------------ | ------------ |
| key | string | 否  | 键名，默认返回所有值 |  |
| value | mixed | 否 | 键值 |  |



### 是否存在某个Session

```php
has(string $name): bool
```

#### 参数说明

| 参数名 | 类型 | 是否必须 | 描述 | 默认值 |
| ------------ | ------------ | ------------ | ------------ | ------------ |
| key | string | 是  | 键名 |  |


### 删除某个Session

```php
delete(string $name): void
```

#### 参数说明

| 参数名 | 类型 | 是否必须 | 描述 | 默认值 |
| ------------ | ------------ | ------------ | ------------ | ------------ |
| key | string | 是  | 键名 |  |


### 清空Session

```php
clear(): void
```
